## Stellar
**Stellar** is a 3D arcade-style space shooter where players pilot highly maneuverable spacecraft through waves of increasingly difficult enemies.  
Fast-paced combat, responsive controls, and a clean sci-fi aesthetic define the experience.

---

## Gameplay

- Pilot a fully controllable 3D spacecraft.
- Dodge enemy shots and survive as long as possible.
- Destroy enemies to increase your score.
- Fight wave after wave of progressively harder opponents.
- Score increases based on eliminated enemies and survival time.
- Game ends when your ship is destroyed.

---

---

## Project Structure

- **Folders:**
  - Scripts
  - Prefabs
  - Materials
  - UI
  - Models
  - Scenes
- **Principles:**
  - Clean, modular C# code in English.
  - Unity best practices (naming, separation of concerns, comments).
  - No default Unity UI or geometry.

---

## Visual Style

- Custom 3D ship models.
- Custom environment, VFX, UI, icons, and skybox.
- Sci-fi aesthetic with glowing HUD elements.
- Smooth feedback for hits, explosions, and movement.

---

## Controls

- **Move:** WASD / Arrow Keys  
- **Ascend:** Space  
- **Descend:** Left Shift  
- **Shoot:** Left Click  
- **Switch Flight Mode:** Ctrl  
- **Pause:** Escape
